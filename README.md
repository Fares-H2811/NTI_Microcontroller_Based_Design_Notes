## 🚀 Getting Started

### Prerequisites
- GCC compiler or any C compiler
- Text editor or IDE (VS Code, Code::Blocks, Dev-C++)
- Basic understanding of programming concepts

### Compilation Example
```bash
# Compile a simple C program
gcc -o program program.c

# Run the program
./program
```

### Running Examples
1. Clone the repository
2. Navigate to the desired session folder
3. Compile and run the example programs
4. Modify and experiment with the code
